## Description

Rust library for reading /etc/{passwd,group,shadow} files.

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
pgs-files = "0.0.7"
```

and this to your crate root:

```rust
extern crate pgs_files;
```
